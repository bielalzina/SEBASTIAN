# TOOLS.md — Notes sobre Eines

Manual d'ús d'eines de SEBASTIAN per la seva pròpia referència.

## Convencions d'Eines
1. Totes les eines han de rebre i retornar JSON.
2. Al terminal, usa `powershell` si el sistema és Windows (permet `vbox`, `ls` vs `dir`, etc.).
3. Els camins han de ser absoluts per evitar confusions si es corre al Docker.

## Llista d'Eines (Planificades)
- `shell`: Executa comandes al host o sandbox.
- `file_ops`: Llegeix, escriu, copia o esborra fitxers.
- `search`: Cerca web (DuckDuckGo).
- `git`: Gestiona el repo del workspace.
- `python_exec`: Corre fragments de codi Python ad-hoc.
