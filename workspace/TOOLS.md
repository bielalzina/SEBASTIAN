# TOOLS.md — Notes sobre Eines

Manual d'ús d'eines de SEBASTIAN per la seva pròpia referència.

## Convencions d'Eines
1. Totes les eines han de rebre i retornar JSON.
2. Al terminal, usa `bash` si el sistema és Linux (Debian 13) o `powershell` si és Windows.
3. Els camins han de ser absoluts per evitar confusions.

## Llista d'Eines Disponibles
- `shell`: Executa comandes al host o sandbox.
- `file_ops`: Llegeix, escriu, copia o esborra fitxers.
- `web_search`: Cerca web (DuckDuckGo).
